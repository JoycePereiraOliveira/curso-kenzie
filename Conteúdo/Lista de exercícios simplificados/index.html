Treinando a propriedade display do CSS

Altere a tag li para ter as seguintes propriedades:

  • Exibir os elementos do li em linha.
  • Borda preta média, com duas linhas.
  • Nenhum tipo de estilo de lista.
  • Fonte com tamanho 3em.

Adicione o valor correto á propriedade, para que os elementos "a" respeitem o espaçamento já atribuído. Algumas opções são:

  • inline
  • inline-block
  • block
  • flex

Adicione o valor correto à propriedade, para que os elementos "a" fiquem alinhados 1 em cima do outro, ocupando todo o espaço disponível. Essas são algumas as opções disponíveis:

  • inline
  • inline-block
  • block
  • flex
